# 🎮 StreamGames - Juegos para TikTok Live

Plataforma de juegos interactivos para streamers de TikTok.

## Instalación

```bash
cd tiktok-stream-games
npm install
npm start
```

Abre http://localhost:3000 en tu navegador.

## Cómo usar

1. Inicia tu live en TikTok
2. Escribe tu @usuario en la barra superior y haz clic en **Conectar**
3. Elige un juego y ponlo como fuente en OBS (Browser Source)

## Juegos disponibles

| Juego | Comando del chat |
|-------|-----------------|
| 🙈 Yo Nunca | `!nunca [frase]` · `!si` · `!no` |
| 🎰 Ruleta | `!entrar` |
| 🧠 Trivia | `A` `B` `C` `D` |
| 🎯 Verdad o Reto | `!verdad [texto]` · `!reto [texto]` |
| 🔢 Adivina el Número | Escribe el número directamente |
| ⚔️ Batalla de Equipos | `!rojo` · `!azul` · `!atacar` |
| 🎭 Mímica | Escribe la respuesta libremente |
| 🃏 Mayor o Menor | `!mayor` · `!menor` |
| 🏃 Carrera de Emojis | Spamea el emoji de tu corredor |
| 🎤 Karaoke Vote | `!1` `!2` `!3`... |

## Requisitos

- Node.js 16+
- Cuenta de TikTok activa en live
